    Smogon Stats API

    Welcome to the Smogon Stats API! You can use the API to access Smogon usage stats for a Pokemon.

    This API is free to use. However, if plan on making a lot of calls, I do ask that you cache the data.

    Data is separated by months, tier, and tier rank. If you are not sure what month or tier rank you would like to access, you can access an endpoint that will provide Pokemon stats on the most recent month and best tier rank, given a tier and Pokemon.

    For example, if you are want stats of Porygon-Z for September 2020 in VGC rank 1760, you would use the former endpoint. If you would like the most recent stats for Porygon-Z in the VGC format, you can use the latter endpoint. Accessing both resources will provide the same stats as rank 1760 is the best rank for VGC and September 2020 is the most recent month.

    Data is available for the following months: July 2020, August 2020, September 2020.

    Stats

    Get Pokemon stats for a specified time, tier, and tier rank

    import requests

response = requests.get('https://smogon-stats.herokuapp.com/stats/2020/09/gen8vgc2020/1760/Togekiss')

data = response.json()
print(data)

    curl "https://smogon-stats.herokuapp.com/stats/2020/09/gen8vgc2020/1760/Togekiss"

    fetch('https://smogon-stats.herokuapp.com/stats/2020/09/gen8vgc2020/1760/Togekiss')
  .then(response => response.json())
  .then(data => console.log(data));

    This endpoint retreives stats for a specific Pokemon given a valid year, month, tier, tier rank, and Pokemon. This example shows how to access VGC rank 1760 stats for Togekiss during September 2020

    HTTP Request

    GET https://smogon-stats.herokuapp.com/stats/<year>/<month>/<tier>/<tier_rank>/<pokemon>

    URL Parameters

    Parameter Description Example
    year The year the data was released "2020" for the year 2020
    month The month the data was released "09" for the month of September
    tier The Smogon tier "gen8vgc2020" for Gen 8 VGC
    tier_rank The rank of the Smogon tier "1760" for the top tier in most formats
    pokemon The name of the Pokemon you want stats for "Togekiss"

    This endpoint retreives the most recent stats for a Pokemon in a specified tier. This request shows how to access the most recent, top stats for Togekiss in the VGC format. This request is equivalent to the first example.

    HTTP Request

    GET https://smogon-stats.herokuapp.com/recent/<tier>/<pokemon>

    URL Parameters

    Parameter Description Example
    tier The Smogon tier "gen8vgc2020" for Gen 8 VGC
    pokemon The name of the Pokemon you want stats for "Togekiss"
